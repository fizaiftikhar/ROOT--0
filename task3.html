Steps to Create a Basic HTML File

  1. Open Notepad or any code editor.
  2. Create a new file.
  3. Write the basic HTML structure.
  4. Save the file with .html extension.
  5. Open the file in a web browser.

When should we use an ordered list?

We use an ordered list when the order of items is important. For example, steps in a process or instructions should be written using an ordered list.

What is the difference between ordered and unordered lists?

An ordered list uses numbers and shows items in a specific sequence, while an unordered list uses bullets and the order of items does not matter.

Where are definition lists useful in real websites?

Definition lists are useful on real websites for displaying glossaries, FAQs, dictionaries, and term explanations where each word needs a clear definition.
