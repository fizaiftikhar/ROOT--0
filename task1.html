LEARNING HTML STRUCTURE

THE CORRECT FILE NAME FOR AN HTML FILE

The correct file name for an HTML file should end with .html.This extension tells the browser that the file contains HTML code.

PORPOSE OF DOCTYPE HTML

The <!DOCTYPE html> declaration tells the browser that the document is written in HTML5.It helps the browser display the page correctly.

Differences between <head> and <body>

1. The <head> tag contains information about the page, while the <body> tag contains visible content.
2. The <head> is not shown on the browser, but the <body> is shown.
3. The <head> includes title and meta tags, while the <body> includes text, images, and links.

What happens if the <head> tag is removed?

If the <head> tag is removed, the browser may still show the page, but important information like title and metadata will not work properly.

Which part of the HTML document is visible on the browser screen, and why?

The <body> part is visible on the browser because it contains all the content such as text, images, headings, and links that users see.

Where is the <title> tag written, and what is its purpose?

The <title> tag is written inside the <head> tag. Its purpose is to show the page title on the browser tab.
