Using the IMG Tag

The alt attribute is important because it provides text information about the image. It helps users who cannot see images, such as screen reader users, and also displays useful text if the image does not load.

Correct Code for Given Folder Structure

Folder structure: Project → index.html → images → profile.jpg

Correct HTML code:

<img src="images/profile.jpg" alt="Profile picture" width="200">

This path works because the image file is inside the "images" folder, which is located in the same directory as index.html. The browser follows the folder path to find and display the image correctly.

What happens if an image file is missing or the path is wrong?

If an image file is missing or the path is incorrect, the browser does not display the image. Instead, it shows a broken image icon. The alt text helps users by showing descriptive text so they can understand what the image was supposed to represent.

Why should images be used carefully on websites?

Images should be used carefully because large images can slow down page loading. Too many images can also affect user experience by making a page cluttered. Proper use of images and alt text is important for accessibility, especially for users who rely on screen readers.

Images in Real Websites

Where are images used?

Images are commonly used in profile pictures, product photos, blog posts, banners, advertisements, and social media posts.

Why are images important for communication, not just design?

Images help explain ideas quickly, show emotions, and make information easier to understand. For example, charts, diagrams, and photos often communicate better than long blocks of text.

When should an image be used instead of text, and when should text be preferred?

Images should be used when visual explanation is necessary, such as showing a product, a diagram, or a real-life example. Text should be preferred when explaining detailed information, instructions, or content that needs to be searchable and accessible.
